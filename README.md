# Auto Hide Tab Bar
A plugin for Krita that auto-hides the tab bar when only a single document is open.

![example](./tab_animation.apng)

How to install:
1. In Krita, choose Tools -> Scripts -> Import Python Plugin from Web...
2. In the prompt, paste: https://github.com/stoofin/krita-auto-hide-tabbar
3. Activate the plugin and restart Krita
